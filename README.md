# CodeFlash - Micro Learning Platform

A lightweight, browser-based application for learning web development concepts through flashcards and quizzes.

## Features

- Responsive design (mobile-first)
- Dark/Light theme support
- Accessible navigation
- Semantic HTML structure
- CSS Grid/Flexbox layouts

## Project Structure
public/
index.html # Homepage
views/
data.html # Lessons listing
form.html # Suggestion form
css/
tokens.css # Design tokens
style.css # Main styles
js/
accessibility.js # Theme & accessibility
data/
db.json # Lesson data
wireframes/ # Design mockups

## Setup

1. Clone repository
2. Serve `public/` directory using local server
3. Open `index.html` in browser

## Technologies

- HTML5 Semantic Elements
- CSS3 with Custom Properties
- Vanilla JavaScript
- JSON data storage

## Accessibility

- WCAG 2.1 AA compliant
- Keyboard navigation support
- Screen reader optimized
- Focus management
